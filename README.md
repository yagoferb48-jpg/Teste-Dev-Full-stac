# Sistema de Cadastro de Pessoas

Este é um projeto simples de CRUD feito para treinar PHP, MySQL e um pouco de JavaScript usando Fetch.  
A ideia é ter um sistema onde é possível cadastrar pessoas, listar, editar e excluir sem recarregar a página.

---

## Tecnologias usadas
- HTML
- CSS
- JavaScript
- PHP
- MySQL

---

## Como funciona
O formulário na tela envia os dados usando Fetch (AJAX) para um arquivo PHP, que faz a parte do CRUD no banco.

O PHP retorna tudo em JSON.  
A página lê esse JSON e atualiza a lista automaticamente.
